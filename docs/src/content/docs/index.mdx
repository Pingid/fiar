## What is fiar

Fiar is a robust library designed to simplify the creation of admin dashboards for Firebase services. It provides essential tools for managing content through Firestore, implementing role-based authentication, and handling asset management via Firebase Storage. Each of these dashboard integrations is highly customizable and extendable to suit your specific needs.

<img {...bannerLight} className="rounded-md border dark:hidden" />
<img {...bannerDark} className="hidden rounded-md border dark:block" />

Fiar is built with flexibility in mind, allowing you to tailor your admin dashboard according to your project's requirements.
You can also extend its capabilities as your project evolves.
